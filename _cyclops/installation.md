# Installation

There are several ways to install and manage Cyclops but the easiest way is by using the [aescripts + aeplugins manager](https://aescripts.com/learn/post/aescripts-aeplugins-manager-app).

## aescripts + aeplugins Manager

1. Download and install the [Manager App](https://aescripts.com/learn/aescripts-aeplugins-manager-app/) from [aescripts + aeplugins](https://aescripts.com/).
2. Log in using your aescripts account to see all purchases.
3. Select Cyclops and press the Install button.
4. To install a trial you can select **Add Trial** from the **Account** menu.
5. Open the Cyclops panel in After Effects using **Window** > **Extensions** > **Cyclops**.


## ZXP Installer

Learn more about installing ZXP files in [this tutorial](https://aescripts.com/knowledgebase/index/view/faq/zxp-installer-faq/) on aescripts + aeplugins.

1. Download and install the [ZXP Installer](https://aescripts.com/learn/zxp-installer/) from [aescripts + aeplugins](https://aescripts.com/).
2. Drag the Cyclops.zxp file into ZXP Installer.
3. Close and restart After Effects.
4. Open the Cyclops panel in After Effects using **Window** > **Extensions** > **Cyclops**.

## Manual Installation

1. Change the extension of the Cyclops.zxp from .zxp to .zip.
2. Unzip this newly created Cyclops.zip to get a folder called Cyclops.
3. Navigate to the Adobe extensions folder.
    - macOS: `/Library/Application Support/Adobe/CEP/extensions`
    - Windows: `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions`
4. Copy the Cyclops folder to the extensions folder.
5. Close and restart After Effects.
6. Open the Cyclops panel in After Effects using **Window** > **Extensions** > **Cyclops**.

{: .warning }
> Manual installation is really only used by extension developers and is offered here only as a last resort.

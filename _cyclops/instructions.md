# Instructions

Cyclops is easy to use, powerful, and fits into any workflow.

Cyclops is different and better than standard screen recording. Screen recording frame rates can be unpredictable which isn’t great when you’ve worked hard to make your animation perfect.

Instead, Cyclops exports a pixel-perfect render that perfectly matches your frame rate and composition dimensions (with [some limitations](/cyclops/faq)).

## Render Workflow
1. Set the Work Area In and Out.
2. Select layers to show the motion paths and layer bounding boxes.
3. Click the **Render** button.
4. Set the save location and file name.

Cyclops will then render a pixel-perfect behind-the-scenes look at your animation. See [Settings](/cyclops/settings) to fine-tune render output, region, type, and speed.

{: .tip }
> For best results fully preview and cache the Work Area In and Work Area Out before using Cyclops.

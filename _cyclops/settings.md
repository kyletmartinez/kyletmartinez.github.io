# Settings

Cyclops offers a few settings to create the perfect render.

## Render Output

Cyclops provides options for both H.264 (MP4) and PNG Sequence renders.

## Render Region

**Composition** is similar to using the Render Queue and will result in a render matching your composition size with no extras.

**Application** can be used to record the entire After Effects interface and is helpful when you need to record layers that move beyond the composition bounds. *This style of [case study](https://thefurrow.tv/project/symptom-checker) was made popular by Seth Eckert, Creative Director at [The Furrow](https://thefurrow.tv/).*

## Render Type

**Standard** is similar to using the Render Queue so it will render your composition panel like usual along with motion paths, bounding boxes, and null layers for any selected layers.

**Wireframe** will result in a unique look with only wireframes along with motion paths, bounding boxes, and null layers for all selected layers. *This style of [case study](https://www.behance.net/gallery/110084511/Acuity-Studios) was made popular by Will Taylor, Creative Director at [Ravie](https://ravie.co/).*

## Speed

Typically using the **Fast (Recommended)** speed is ideal. However, more complex compositions with many layers or render intensive effects may benefit from using the **Slow** speed.

This setting simply provides more time for After Effects to render each frame and for Cyclops to capture that frame.
